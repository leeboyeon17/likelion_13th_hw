Template

  1. HTML 페이지 만들기
    • main/templates/main 경로의 폴더에 mainpage.html 파일 생성 (mainpage를 만들 경우)
    • 해당 파일에 웹페이지 내용 구성에 필요한 코드 붙여넣기
  2. HTML 페이지와 View 연결하기
    • views.py 작성: 원하는 url을 통해 작성한 mainpage가 보여지도록 하기 위함
    • urls.py 작성: url을 통해 HTML 페이지와 View를 연결하기 위함
  3. HTML 페이지와 View 연결하기
    • HTML 페이지에서 url에 연결되어 있는 페이지로 이동할 수 있게 하기 위함
    • .html 페이지의 nav에 {} 안에 '% url 'name' %'입력
    • 이때 {} 안 '% url 'name' %'은 Template 언어를 사용한 url 문법이다!

Template 언어란?

    : 파이썬의 변수, 문법을 html에서 사용할 수 있도록 제공하는 직관적인 언어(크게 4가지 기능을 제공)

  1. 템플릿 변수: view에서 생성한 데이터를 변수로써 사용 가능
  2. 템플릿 태그: for문(반복문), if문(조건문) 등을 사용 가능
  3. 템플릿 필터: 템플릿 변수를 특정 서식에 따라 변환 가능
  4. 템플릿 주석: html 내용 중 주석(숨김)이 필요한 경우 사용

효율적인 코드

  • template 상속
    1. project(폴더)>templates(폴더)>base.html(파일) 경로로 생성
    2. 반복되는 코드를 base.html로 이동
    3. 다른 페이지에서 사용될 때를 위해 {} 안에 '% block content %' 와 '% endblock %' 입력
    4. .html 파일에 {} 안에 '% extends 'base.html' %', '% block content %', '% endblock %' 입력하여 base.html과 연결
    5. project>settings.py에서 os를 import 하고 템플릿 파일을 찾을 수 있는 경로 추가
  • nav bar 분리
    1. templates 폴더 내에 shared 폴더 생성
    2. shared 폴더 내에 _navbar.html 파일 생성
    3. base.html에 포함된 nav 영역을 _navbar.html로 옮기기
    4. 분리한 _navbar.html을 base.html에 상속 시키기
